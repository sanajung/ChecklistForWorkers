업무 체크리스트

프로젝트별 단계 진행 상황과 마감일을 한 화면에서 관리하세요.

캘린더
접기
아직 프로젝트가 없어요 "+ 새 프로젝트" 버튼을 눌러 첫 프로젝트를 만들어보세요.
폴더를 선택하면 하위 폴더/파일 구조를 읽어옵니다. 각 항목에 메모를 남길 수 있고, 엑셀(.xlsx) 파일은 이 프로그램의 체크리스트로 바로 불러올 수 있어요. (PC 크롬/엣지 전용)

사용법 안내

프로젝트 → 단계 → 체크 항목 3단 구조로 업무를 관리하는 도구예요.

1. 프로젝트 만들기

  • "+ 새 프로젝트"를 눌러 이름을 입력해요. (예: 신규 툴 개발, 3분기 행사 준비)
  • 기존 프로젝트의 항목을 그대로 복사해서 시작할 수도 있어요 — 반복 업무에 유용해요.

2. 단계와 체크 항목

  • "+ 새 단계 추가"로 업무 흐름을 나눠요. (예: 기획 → 준비 → 실행 → 마무리)
  • 각 단계에는 두 종류의 리스트가 있어요: 할 일 체크리스트(해야 하는 작업)와 실수 방지 체크리스트(놓치기 쉬운 확인 사항).
  • 항목 옆 "상세 ▾"를 누르면 시작일·마감일·메모를 입력할 수 있어요. 메모에는 일을 구체적으로 어떻게 할지 적어두면 좋아요.

3. 잠금 (삭제 방지)

  • 단계 제목 옆 자물쇠(🔓)를 누르면 잠금(🔒) 상태가 돼요.
  • 잠긴 단계는 실수로 삭제할 수 없어요. 체크와 날짜·메모 수정은 계속 가능해요.

4. 정리 기능

  • 단계 왼쪽 ⠿ 핸들을 드래그하거나 ▲▼ 버튼으로 단계 순서를 바꿀 수 있어요.
  • 단계별 색상 버튼을 눌러 색을 지정하면 캘린더와 진행 바에도 같은 색으로 표시돼요.
  • "완료 숨기기"를 켜면 이미 끝낸 항목을 감춰 남은 일에 집중할 수 있어요.

5. 캘린더

  • 단계 마감일과 항목별 시작일·마감일이 자동으로 캘린더에 점으로 표시돼요. (진한 점 = 마감, 옅은 점 = 시작)
  • 날짜를 누르면 그날의 항목들이 아래에 표시되고, 거기서 바로 체크할 수도 있어요.
  • "접기"를 누르면 캘린더를 숨겨 체크리스트에 집중할 수 있어요.

6. 칸반보드 · 간트차트 · 폴더 탐색

  • 칸반보드: 모든 체크 항목이 "할 일 / 진행 중 / 완료" 카드로 표시돼요. 카드를 드래그해서 상태를 바꿀 수 있어요.
  • 간트차트: 시작일·마감일이 입력된 항목이 단계 색상의 막대로 타임라인에 표시돼요.
  • 폴더 탐색: PC의 폴더를 선택하면 하위 구조를 읽어와 폴더·파일마다 메모를 남길 수 있어요. 폴더 안의 엑셀(.xlsx)은 버튼 한 번으로 체크리스트에 불러올 수 있어요. (PC 크롬/엣지 전용)

7. 검색·백업·반복

  • 상단 검색창으로 항목/메모를 찾고, "마감임박"·"지연" 필터로 급한 일만 볼 수 있어요.
  • 맨 위 요약 바에서 지연·오늘 마감·3일 내 건수를 한눈에 확인해요.
  • "백업(JSON)"으로 전체 프로젝트를 파일로 저장하고, 다른 PC/폰에서 "복원"으로 불러올 수 있어요. 주기적 백업을 권장해요.
  • 항목 상세에서 반복(매일/매주/매월)을 설정하면, 완료 시 다음 일정으로 자동 재생성돼요.
  • 항목 옆 작은 ▲▼로 항목 순서를 조정할 수 있고, 간트차트에서는 막대를 좌우로 드래그해 일정을 통째로 옮길 수 있어요.

8. 티켓 상세 (Jira 스타일)

  • 체크리스트 항목의 ✎ 버튼, 칸반 카드 클릭, 간트차트 항목 이름 클릭으로 상세 편집창이 열려요.
  • 제목(이름 수정), 상태, 우선순위, 담당자, 시작/마감일, 반복, 설명을 한 화면에서 편집할 수 있어요.
  • 폴더 탐색에서 파일/폴더의 "연결" 버튼으로 항목과 연결하면 체크리스트에 📎 배지가 표시되고, 상세창에서 목록을 확인·해제할 수 있어요.

9. 반복되는 프로젝트 관리

  • 매 회차 반복되는 프로젝트(예: 분기별 행사)는 "+ 새 프로젝트 → 기존 프로젝트 항목 복사 → 날짜 초기화"로 새 회차를 만드는 걸 권장해요. 항목 구성·담당자·메모는 유지되고 체크와 날짜만 리셋돼요.
  • 지난 회차는 삭제하지 말고 남겨두면 기록·회고 자료가 돼요.
  • 회차 안에서 매주 반복되는 개별 업무는 항목의 "반복" 설정을 쓰세요.

10. 안전장치 · 편의 기능

  • 항목/단계를 삭제하면 하단 알림의 "실행 취소"로 6초 안에 되돌릴 수 있어요.
  • 백업한 지 7일이 지나면 상단에 백업 알림이 떠요.
  • 완료 후 30일 지난 항목은 "보관함"으로 자동 이동돼 목록이 가벼워져요. 보관함에서 언제든 복원할 수 있어요.
  • 프로젝트 선택 목록에 전체 진행률(%)이 표시되고, 칸반보드는 담당자·우선순위로 필터링할 수 있어요.
  • 모바일에서는 칸반 카드의 ◀▶ 버튼으로 상태를 바꿀 수 있어요.

11. 엑셀

  • 엑셀 다운로드: 현재 프로젝트 전체를 엑셀 파일로 저장해요. 백업이나 공유용으로 좋아요.
  • 엑셀 연동(자동저장): 파일 위치를 한 번 지정하면 수정할 때마다 자동 반영돼요. (PC 크롬/엣지 전용)
  • 엑셀 불러오기: 저장해둔 엑셀 파일에서 체크리스트를 복원해요.

데이터 저장 위치

데이터는 이 브라우저에 저장돼요. 브라우저 데이터를 삭제하면 사라질 수 있으니, 중요한 프로젝트는 주기적으로 엑셀 다운로드로 백업하는 걸 추천해요.

보관함

완료 후 30일이 지난 항목이 자동으로 이동돼요.
